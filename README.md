# Classify API

This project is a Node.js and Express application that does the following:
1.  Accepts a name.
2.  Fetches data from Genderize, Agify, and Nationalize APIs.
3.  Classifies the data based on age and nationality.
4.  Stores the classified profile in a database.
5.  Provides API endpoints to create, retrieve, and delete profiles.

## Installation

```bash
npm install
```

## Running the application

```bash
npm start
```
